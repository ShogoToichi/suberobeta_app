# Windows shortcht list

- `Ctrl Shift Esc` タスクマネージャの起動
- `Win Shift S` 画面切り取り
- `Win Alt R` 画面録画即時開始
- `Win P` 画面モード(複製，拡張等)切り替え
- `Win E` エクスプローラーの起動
- `Win R` ファイル名を指定して実行を開く
- `Win L` コンピューターをロック
- `Win I` 設定を開く
- `Win {num}` タスクバーの{num}番目のアプリを開く，Shift をつけると新しいウインドウで開く
- `Win X` Win アイコンの右クリック
